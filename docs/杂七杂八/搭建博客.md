## [hexo](https://hexo.io/zh-cn/)

### 安装前提

先安装下列应用程序即可：

- [Node.js](http://nodejs.org/)
- [Git](http://git-scm.com/)

### 安装

```bash
# 安装 Hexo
npm install hexo-cli -g

hexo init 文件名
cd 文件名
npm install

# 启动服务器
hexo server
```
